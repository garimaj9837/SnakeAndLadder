# Snake And Ladder

A Low-Level Design (LLD) implementation of the classic Snake and Ladder game using Java and Object-Oriented Design principles.

## Features

- Multiple players support
- Configurable snakes and ladders
- Dice rolling mechanism
- Automatic snake and ladder traversal
- Winner detection
- Clean object-oriented design

## Design

### Main Classes

- `Game` - Controls game flow
- `Board` - Maintains snakes, ladders, and board size
- `Player` - Represents a player
- `Dice` - Generates dice rolls
- `Snake` - Defines snake start and end positions
- `Ladder` - Defines ladder start and end positions

## Technologies Used

- Java
- Collections Framework
- SOLID Design Principles
